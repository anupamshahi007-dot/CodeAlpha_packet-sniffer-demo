# 🤝 Contributing

Thank you for your interest in contributing to this project!

This repository was created as an educational project to help beginners learn Python, networking, and packet analysis.

Contributions that improve the project, documentation, or learning experience are welcome.

---

## Ways You Can Contribute

You can help by:

- Fixing bugs
- Improving documentation
- Adding comments to the code
- Optimizing existing code
- Creating new packet analysis examples
- Improving error handling
- Adding new protocol support
- Suggesting new features

---

## Before You Submit

Please make sure that:

- Your code is well documented.
- You follow Python best practices.
- Existing functionality is not broken.
- Changes are tested before submitting.

---

## Reporting Issues

If you discover a bug or have a suggestion, please open a GitHub Issue and include:

- A clear description
- Steps to reproduce the issue
- Expected behavior
- Actual behavior
- Screenshots or logs (if applicable)

---

## Code Style

Please keep code:

- Clean
- Readable
- Well commented
- Beginner-friendly

---

## Code of Conduct

Please be respectful and constructive when interacting with others.

---

Thank you for helping improve this project! 🚀
