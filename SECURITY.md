# 🔒 Security Policy

Thank you for helping keep this project secure.

This repository is an educational cybersecurity project designed to demonstrate Python-based network packet capture and protocol analysis.

## Supported Versions

The latest version of this repository receives updates and improvements.

| Version | Supported |
|---------|-----------|
| Latest | ✅ Yes |
| Older versions | ❌ No |

---

## Reporting a Security Issue

If you discover a security issue related to this project, please:

1. Open a GitHub Issue if the report is suitable for public discussion.
2. Clearly describe the issue.
3. Include steps to reproduce it, if possible.
4. Suggest a possible fix if you have one.

Please avoid sharing sensitive information (passwords, private keys, personal data, or confidential network captures) in public issues.

---

## Responsible Use

This project is intended for:

- Learning Python networking
- Understanding network protocols
- Educational demonstrations
- Authorized network analysis
- Ethical cybersecurity practice

Do **not** use this software to monitor networks or devices without permission. Unauthorized packet capture may violate laws, regulations, or organizational policies.
